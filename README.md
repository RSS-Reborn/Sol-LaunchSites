# Sol-LaunchSites
Sol Textures for various Launchsites around Earth.

Contains the following launchsites (more will be added later)
- Woomera, Australia
- Churchill, Canada
- Jiuquan, China
- Taiyuan, China
- Wenchang, China
- Xichang, China
- Hammaguir, Algeria
- Kourou, French Guiana
- Palmachim, Israel
- Satish Dhawan, India
- Semnan, Iran
- Tanegashima, Japan
- Uchinoura, Japan
- Sohae, North Korea
- Naro, South Korea

There are 3 different resolution options available, 256x, 512x, 1024x.
These values represent the width in pixels of a lat-long tile.

256x  - 434.4m/px
512x  - 217.2m/px
1024x - 108.6m/px
